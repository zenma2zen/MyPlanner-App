# 🚀 MyPlanner App
---

## Fitur Utama

* **Interface Estetik & Clean:** Desain UI yang super bersih, modern, dan nyaman di mata (Apple-inspired aesthetic).
* **Manajemen Tugas:** Bikin *To-Do List*, catat agenda harian, dan pantau produktivitasmu dengan mudah.
* **Performa Ringan:** Aplikasi gesit, responsif, dan siap menemani aktivitas harianmu.

---

## Cara Instalasi

1. Download file APK versi terbaru di halaman [Releases](../../releases).
2. Install di perangkat Android kamu (aktifkan izin instalasi dari sumber tidak dikenal jika diperlukan).
3. Buka aplikasi dan mulai atur harimu agar lebih teratur!
